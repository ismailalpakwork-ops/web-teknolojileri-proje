# LynxMovie | İnteraktif Medya Kitaplığı
Bu proje, ISE-201 Web Teknolojileri dersi kapsamında geliştirilmiş bir Single Page Application (SPA) uygulamasıdır. Kullanıcıların film ve dizi verilerine erişebilmesini, filtreleme yapabilmesini ve favorilerini saklayabilmesini sağlar.
---

## 🚀 Özellikler


Dinamik Veri: Veriler asenkron olarak bir yerel JSON dosyasından çekilmektedir.



Gelişmiş Filtreleme: Medyalar isme, kategoriye ve yıla göre anlık olarak filtrelenebilir.


SPA Yapısı: Detay sayfası ayrı bir HTML dosyası olmadan, JavaScript ile dinamik olarak yönetilir.


Favorilerim: localStorage kullanılarak seçilen medyalar tarayıcıda saklanır.



Duyarlı Tasarım: Uygulama; mobil, tablet ve masaüstü cihazlarla tam uyumludur.


### 🛠️ Kullanılan Teknolojiler

HTML5: Anlamsal (Semantic) etiket yapısı.



CSS3: Flexbox ve Grid sistemleri.



Modern JavaScript: ES6+ (fetch API, async/await, localStorage).
